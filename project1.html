DRY Principle • BEM Methodology • Separation of Concerns

♻️

DRY Principle

Don't Repeat Yourself. One component defined once, used everywhere.

.button { padding: 1rem; border-radius: 4px; } .button--primary { background: #2563eb; } .button--secondary { background: #1e293b; }
🧩

BEM Methodology

Block • Element • Modifier

.card { } .card__header { } .card--featured { }

Fractal scale: modular, scalable, maintainable.
